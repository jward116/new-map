# Version 2.3 Field Info Upgrade

## Goal
Make the app feel like a real field reference tool, not just a map with an outline.

## Add field status details
- Inside/outside boundary
- Kansas or Nebraska
- Estimated county
- GPS accuracy
- Lat/lng
- Nearest known public place
- Distance to nearest place
- Correct parcel source button
- Copy full field summary button

## Improve public place data
Each place should have:
- name
- category
- address
- county
- state
- phone
- website
- notes
- latitude
- longitude
- public/private flag

## Categories
- Law Enforcement
- Tribal Government
- Court
- Casino / Business
- Public Facility
- Road / Access
- Agriculture / Farm
- Utility / Maintenance
- Other

## Public safety/contact section
Add public non-sensitive contact info:
- Doniphan County Sheriff's Office
- Brown County Sheriff's Office
- Richardson County Sheriff's Office
- Nearby dispatch/non-emergency contacts
- Fire/EMS public contacts

## Do not add to public GitHub Pages
- Gate codes
- Private residence notes
- Investigation notes
- Confidential contacts
- Officer-only intelligence
- Evidence info
