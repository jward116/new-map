# ITKN Field Map - Version 2 Plan

## Goal
Turn the working phone map into a more useful field tool.

## Keep working from v1
- Live GitHub Pages app works
- Phone GPS works
- Public boundary loads
- State line loads
- Public ITKN places load
- Parcel source links work
- Saved field points work

## Version 2 upgrades

### 1. Mobile layout
- Replace large phone panel with bottom drawer
- Keep map full-screen
- Add simple tabs:
  - Status
  - Nearby
  - Parcels
  - Save
  - Layers

### 2. Better location status
Show:
- Inside/outside loaded boundary
- Kansas or Nebraska
- Approximate county
- GPS accuracy
- Lat/lng
- Nearest known public place
- Distance to nearest place
- Copy coordinates button
- Open Google Maps button
- Open correct county parcel link button

### 3. More public data
Add:
- Tribal public facilities
- Public tribal businesses
- County sheriff/dispatch contacts
- Fire/EMS public contacts
- County parcel/assessor links
- Public notes only

### 4. Field point categories
Add save-point types:
- Gate
- Pasture
- Business
- Tribal property
- Road issue
- Animal/livestock call
- Boundary question
- Contact location
- Other

### 5. Import/export
Improve:
- Export saved points as GeoJSON
- Import saved GeoJSON points back into the app
- Clear all saved test points button

## Do not add to public GitHub Pages
- Gate codes
- Private residence notes
- Investigation notes
- Confidential contacts
- Officer-only intelligence
- Evidence information
